⚠️ COMMON ISSUES AND SOLUTIONS
================================================================================

ISSUE: "py is not recognized as an internal or external command"
SOLUTION: Python is not installed or not in PATH
  Fix 1: Use "python" or "python3" instead
  Fix 2: Download Python from python.org
  Fix 3: Reinstall Python and check "Add Python to PATH"

ISSUE: "No such file or directory" or "File not found"
SOLUTION: Not in correct directory
  Fix: Type: cd D:\calculator\calculator
       Then: py final_test_setup.py

ISSUE: "Access is denied"
SOLUTION: Permission issue
  Fix: Run Command Prompt as Administrator
       Windows + R → cmd → Ctrl+Shift+Enter

ISSUE: "FileNotFoundError: calculator.py"
SOLUTION: Script is in wrong directory
  Verify: You are in D:\calculator\calculator
  Check: dir calculator.py

ISSUE: Script runs but no output
SOLUTION: Output was too fast to see
  Fix: Redirect output to file: py final_test_setup.py > output.txt
